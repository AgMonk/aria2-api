# 引入

```xml

<project>
    <repositories>
        <repository>
            <id>com.github.gin.utils</id>
            <name>aria2-api</name>
            <url>https://agmonk.github.io/aria2-api/maven-repo/</url>
        </repository>
    </repositories>

    <dependencies>
        <dependency>
            <groupId>com.github.gin.utils</groupId>
            <artifactId>aria2-api</artifactId>
            <version>1.0.0</version>
        </dependency>
    </dependencies>

</project>
```
